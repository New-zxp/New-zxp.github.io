stdio.h头文件

stdio.h 是 C 语言标准库的一部分，它提供了与标准输入（keyboard、文件等）和标准输出（屏幕、文件等）相关的函数和宏定义。这个头文件允许您在 C 程序中进行输入和输出操作，以及文件处理。标准库中包含了众多函数，如 printf 和 scanf，用于格式化输出和输入，还有文件操作函数，如 fopen 和 fclose，用于打开和关闭文件等。

通过包含 stdio.h 头文件，您可以在程序中使用这些函数，使得您能够与用户进行交互、读取和写入文件，以及执行各种输入输出操作。

%d 是格式说明符，用于表示要插入到字符串中的整数值。在这里，%d 表示将整数 x 的值插入到字符串中。

x 是要插入到格式化字符串中的整数值，它将替代 %d。

"\n" 表示一个换行符，用于在输出中换行，使输出更易于阅读。

顺序表

#define MaxSize 10: 这行代码使用宏定义了一个常量 MaxSize，它的值为10。这个常量用来表示顺序表的最大容量，即顺序表中可以存储的元素的最大数量。

#define 是C语言和C++语言中的一个预处理指令，它用于创建宏定义（Macro Definition）。宏定义是一种文本替换机制，它允许你为一段代码片段或值创建一个符号名，当代码中出现这个符号名时，预处理器会在编译之前将其替换为相应的定义。

typedef struct { ... } SqlList;: 这行代码定义了一个名为 SqlList 的自定义数据类型，这个类型是一个结构体。结构体中包含两成员：

  • ElemType data[MaxSize];: 这是一个数组，用来存储顺序表中的元素。ElemType 表示元素的数据类型，通常在实际使用时会用具体的数据类型来替代，例如 int、char、或者其他自定义数据类型。数组的长度由前面定义的 MaxSize 决定，最大为10。
  • int length;: 这是一个整数，用来记录当前顺序表中已经存储的元素的数量。在顺序表中，这个值通常会随着插入和删除操作的进行而动态改变。

typedef 是C和C++编程语言中的关键字，用于为现有的数据类型创建用户定义的数据类型别名。这是一个非常有用的功能，它可以提高代码的可读性、可维护性，同时让代码更加清晰和易于理解。typedef 的主要用途包括：

  1. 创建别名: 通过 typedef，你可以为已经存在的数据类型（如 int、float、struct、enum等）创建一个新的名字，使代码更易于理解。这种别名可以让你在代码中使用更具描述性的标识符。

  2. 增加代码的可移植性: 通过使用 typedef，你可以为特定的数据类型创建别名，这有助于提高代码的可移植性，因为你只需在一个地方修改类型别名，而不必在整个代码中寻找和替换具体的数据类型。

  3. 简化复杂类型: 在C和C++中，复杂的数据类型，如结构体、指针或函数指针类型，可以使用 typedef 简化，使代码更加易读。

  4. 提高可维护性: 使用 typedef 可以提高代码的可维护性，因为它允许你在一个地方定义和修改数据类型的别名，而不必在整个代码中寻找和替换具体的数据类型。

stdlib.h头文件

#include <stdlib.h> 是C语言标准库的一个头文件，它包含了一些标准函数和宏，用于执行与内存分配、进程控制、随机数生成、字符串转换和其他底层系统操作相关的任务。以下是一些 stdlib.h 头文件中包含的常见函数和宏：

  1. Memory Allocation Functions:
    • malloc(): 用于动态分配内存。
    • calloc(): 用于分配指定数量的内存块，并将它们初始化为零。
    • realloc(): 用于更改先前分配的内存块的大小。
    • free(): 用于释放之前分配的内存。
  2. Process Control Functions:
    • exit(): 用于退出程序并返回一个状态码。
    • atexit(): 用于注册在程序终止时自动调用的函数。
    • system(): 用于执行系统命令。
  3. Random Number Functions:
    • rand(): 用于生成伪随机整数。
    • srand(): 用于设置随机数生成的种子值。
  4. String Conversion Functions:
    • atoi(), atol(), atof(): 用于将字符串转换为整数或浮点数。
    • itoa(), ltoa(): 用于将整数转换为字符串。
  5. Other Utility Functions:
    • abs(): 用于计算整数的绝对值。
    • labs(): 用于计算长整数的绝对值。

stdlib.h 头文件提供了这些函数和宏的声明，以及它们的相关定义，以便在C程序中使用。这些函数和宏是C语言的标准库的一部分，因此它们在各种C编译器和操作系统中都是可用的。

十六进制和二进制转换

十六进制转二进制

  1. 将每个十六进制数字转换为其对应的四位二进制数。
  2. 按顺序连接所有的二进制数位，得到最终的二进制表示。

举个例子，将十六进制数 0xABCD 转换为二进制数：

  1. 0xA 对应的二进制数：1010
  2. 0xB 对应的二进制数：1011
  3. 0xC 对应的二进制数：1100
  4. 0xD 对应的二进制数：1101

因此，0xABCD 对应的二进制数为：1010101111001101。

二进制数转换为十六进制数

  1. 将二进制数从右向左每四位一组进行分组。
  2. 将每组四位二进制数转换为其对应的十六进制数字。

举个例子，将二进制数 1010101111001101 转换为十六进制数：

  1. 1010 对应的十六进制数：A
  2. 1011 对应的十六进制数：B
  3. 1100 对应的十六进制数：C
  4. 1101 对应的十六进制数：D

因此，1010101111001101 对应的十六进制数为 0xABCD。

猜数字游戏

  • srand() 函数用于设置随机数生成器的种子。它的参数是一个整数，被称为种子值。
  • 通常情况下，为了确保每次运行程序时都能产生不同的随机数序列，我们会使用 time(0) 函数获取当前时间作为种子值。这样可以保证每次程序运行时种子值都不同。
  • 因此，srand(time(0)); 会将当前时间作为种子值传递给随机数生成器，以确保每次运行程序时都产生不同的随机数序列。
  • rand() 函数用于生成伪随机数。它不需要参数，每次调用都会返回一个看似随机的数值。
  • 在调用 rand() 函数之前，通常需要先调用 srand() 函数来设置随机数种子。否则，如果不设置种子，每次程序运行时将会生成相同的随机数序列。

  • scanf()：输入

  • rand()：用于生成伪随机整数。

  • srand()：用于设置随机数生成的种子值。

  • sizeof()：计算数据类型在内存中占用的空间

    在C语言中，a 是一个数组，而 *p 是一个指针。尽管数组名和指针在某些情况下可以互换使用，但它们在 sizeof 运算符中的行为是不同的。

    • sizeof(a)：这个表达式计算的是整个数组 a 在内存中所占的字节数。因为 a 是一个包含10个整型元素的数组，每个整型元素通常占用4个字节（具体大小可能取决于编译器和操作系统），所以 sizeof(a) 的结果通常是 10 * sizeof(int)，即40。

    • sizeof(*p)：这个表达式计算的是指针 p 所指向的对象的大小。因为 p 指向整型元素，所以 sizeof(*p) 的结果通常是 sizeof(int)，即4。

    • sizeof(*a)：这个表达式计算的是数组 a 中第一个元素的大小。因为 *a 表示数组中的第一个元素，所以 sizeof(*a) 的结果通常是 sizeof(int)，即4。

    因此，虽然 sizeof(*p) 和 sizeof(*a) 都是取得数组或指针所指向的元素的大小，但 sizeof(a) 是整个数组 a 的大小，因此它们的结果不同。

调用函数时给的值和参数类型不匹配是c语言传统上最大的漏洞，编译器总是悄悄地替你把类型转换好，这可能不是你所期望的

变量的地址

在C语言中，通常会以十六进制的形式打印地址，使用十六进制可以更方便地观察和比较地址的值，因为它们与内存地址的字节对应。

地址的单位通常是字节，内存被划分为一个个字节，每个字节都有唯一的地址。

a的地址是0xbff81d6c
b的地址是0xbff81d68
这表明a和b存储在内存中相邻的位置上，但是a的地址比b的地址高4个字节。在许多情况下，编译器会尽量将相邻的变量分配在相邻的内存地址上，以便提高内存访问的效率。这种情况下，a和b很可能被分配在栈上，栈是一种用于存储局部变量和函数调用信息的内存区域。在典型的32位系统中，整型变量int通常占据4个字节的内存空间。因此，a和b之间的地址差异正好是一个整型变量的大小。

之所以不能写出scanf("%d", i);这样的形式，是因为scanf是根据地址来改变变量的值，直接用i的话会导致，输入的值会改变i的值对应的地址（也就是在内存中地址为6的变量）的值

数组变量是特殊的指针

数组变量本身表达地址

void test(int *a)和void test(int a[])是等价的

  • ```c
    int a[10];
    int *p = a;//可以不用&取地址

  • 数组a的地址和a[0]的地址是相等的a == &a[0]

  • []运算符可以对数组做也可以对指针做

    p[0] <==> a[0]

  • *运算符可以对指针做，也可以对数组做

    *a = x，x是a[0]的值

  • 数组变量是const的指针，所以不能被赋值

    int b[] --> int * const b

    b = a，是不行的，因为b是一个特殊的const(常量)指针，它的值也就是数组的地址是不能被改变的，因此不能让b=a，让b的值（也就是数组b的地址）指向a的地址

指针与const

  • int * const q = &i;

    *q = 25//OK

    q++//Error

    表示指针一旦得到了某个变量的地址，不能再指向其他变量

  • const int * p = &i;

    *p = 6;//error

    i = 26//ok

    p = &j//ok

    表示不能通过这个指针去修改那个变量（并不能使那个变量成为const）

  • 判断哪个被const了的标志是const在*前面还是后面

    前两个表示不能通过这个指针修改变量

    最后一个表示指针不能指向其他变量

字符串

字符串变量的定义

字符数组：char word = {'H','E','L','L','O'}

字符串：char word = {'H','E','L','L','O','\0'}

  • 字符串是以’\0’结尾的一串字符

  • ‘\0’标志字符串的结束，但是他不是字符串的一部分，计算字符串长度时不包含这个0

  • 字符串是以数组的形式存在，以数组或指针的形式访问

    更多是以指针的形式

  • string.h里有很多处理字符串的函数

  • "hello"会被编译器变成一个字符数组放在某处，这个数组的长度是6，结尾还有表示结束的0

字符串常量

  • s是一个指针，初始化为一个指向字符串常量

  • 由于这个常量所在的地方，实际上s是const char* s，但是由于历史原因编译器接收不带const的写法

  • 但是试图对s所指的字符串做写入会造成严重的后果

  • 如果需要修改字符串应该由数组定义

    char s[] = "hello,world"

选择指针还是数组

  • 数组：这个字符串在这里

    作为本地变量空间自动被回收

  • 指针：这个字符串不知道在哪

    • 处理参数
    • 动态分配空间

  • 如果要构造一个字符串，选择数组

  • 如果要处理一个字符串，选择指针

字符串赋值

并没有产生新的字符串，只是让指针s指向了指针t所指向的字符串，对s的操作就是对t做的

字符串数组

  • char **a

    这个a是一个指向指针的指针，被指向的指针指向一个字符串

  • char a[][]，第二个[]必须有明确的值

    例如a[][10]，表示字符串数组中每个字符串的长度不超过10

  • char *a[]

    表示一个指针数组，里边的指针分别指向一个字符串

字符串函数

  • strlen

    size_t strlen(const char *s)

    返回s字符串长度，不包含’\0’

  • strcmp

    int strcmp(const char *s1,const char* s2)

    比较两个字符串，返回

    • 0：s1 == s2
    • 1：s1 > s2
    • -1：s1 < s2

  • strcpy

    char *strcpy(char* restrict dst,const char *restrict src)

    • 把src的字符串拷贝到dst
    • restrict表示src和dst不重叠
    • 返回dst，为了使代码能链起来

  • strchr
